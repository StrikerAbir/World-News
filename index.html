Var: We can use it anywhere. It has some problems too. We can redeclare it. It can be an accidental global variable. That's why to avoid problems they created two other variable systems. Let and Const.

Let: It allows that variable to change. Just like var. But it s ES6 version and have some difference.

Const: It doesn't allow to change of variables. Making variable constant.

Map() function: this function only works on array. map() function have call back function, that execute on every element of an array. And it returns a new array.

Foreach() function: this function only works on array. foreach() function have call back function, that execute on every element of an array. And it doesn't returns any array.

Filter() function: this function only works on array. Same type of map. It returns an array. Its call back function have conditions, when condition is true it returns value.

Find() function: this function only works on array. Same type of filter. It returns an array. Its call back function have conditions, when condition is true it returns only first value that true.
